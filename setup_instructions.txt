Disease Outbreak Prediction System - Setup Instructions

Prerequisites:
- Python 3.8+ installed
- Node.js and npm installed

1. Starting the Backend (Flask)
-------------------------------
1. Open a terminal and navigate to the backend directory:
   cd project\backend

2. (Optional but recommended) Activate the virtual environment if not already activated.
   From the root of the project (if you are in the project folder):
   ..\venv\Scripts\activate
   Or if you are at the repository root:
   venv\Scripts\activate

3. Install the required Python dependencies:
   pip install -r requirements.txt

4. Start the Flask server:
   python app.py

The backend will start running, typically on http://127.0.0.1:5000.

2. Starting the Frontend (React / Vite)
---------------------------------------
1. Open a NEW terminal and navigate to the frontend directory:
   cd project\frontend

2. Install the required Node.js dependencies:
   npm install

3. Start the development server:
   npm run dev

The frontend will start running. Check the terminal output for the local URL (typically http://localhost:5173).

3. Accessing the Application
----------------------------
Open your web browser and go to the frontend URL provided in step 2 (e.g., http://localhost:5173).
The backend API is running concurrently and will be accessed by the frontend to fetch data and predictions.
